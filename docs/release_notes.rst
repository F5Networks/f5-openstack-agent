Release Notes v |release|
=========================

This release provides an implementation of the F5® agent to support the use of F5 Networks® BIG-IP® systems with OpenStack Neutron LBaaSv2.

Release Highlights
------------------

This release introduces the following:

- Hierarchical Port Binding (integration with Cisco ACI)
- Bug fixes

See the `changelog <https://github.com/F5Networks/f5-openstack-agent/compare/v9.2.0...v9.3.0.b2>`_ for the full list of changes in this release.
